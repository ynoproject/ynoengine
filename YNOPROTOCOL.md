## ynoproject protocol specifications



#### Basic Data Types:

| Name   | Description                     |
| ------ | ------------------------------- |
| int32  | 4-byte                          |
| uint32 | 4-byte                          |
| int16  | 2-byte                          |
| uint16  | 2-byte                          |
| byte   | 1-byte                          |
| bool | 1-byte |
| string | length (uint32) + following data |
| uuid | 16-byte |



#### S2C (Server to Client) Packets:

SYNC_PLAYERDATA:

| Data Field  | Type   |
| ----------- | ------ |
| host_id     | int32  |
| key         | string |
| uuid        | uuid   |
| rank        | int32  |
| account_bin | int32  |
| badge       | string |
